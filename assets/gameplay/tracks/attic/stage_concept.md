# Attic Stage Concept

Stage: Attic / Popper  
Faction: The Attic Pranks  
Reference image: `stage_key_art.png`

## Identity

The Attic is pristine but uncanny. Popper's faction has converted boxes, strings, trunks, sheets, hidden ramps, false gates, and prank mechanisms into a rule-breaking race space.

The room should not look neglected. It should look clean and arranged in a way that makes players distrust what they see.

## Lore Purpose

The Attic Pranks reject all stable authority. Popper uses the attic to prove every rule can be cheated, every route can hide another route, and every ruler can be mocked.

The stage teaches the appeal and cost of freedom without trust.

## Visual Read

- Clean attic storage: boxes, trunks, sheets, beams, string mechanisms, tidy stacks.
- Toy-scale route language: false gates, trunk maze, sheet tunnel, box switchbacks, hidden ramps.
- Faction modifications: prank triggers, misdirection arrows, fake finish shapes, dangling strings.
- Mood: playful, deceptive, uncanny, clever, unstable.

## Route Fantasy

The race should feel like the track is lying. Drivers enter trunk mazes, choose between real and false gates, duck through sheet tunnels, dodge marble traps, and discover hidden ramps that cut across expected routes.

Shortcuts should reward memory and nerve, but never become unreadable random punishment.

## Hero Landmarks

- Prank trunk maze.
- String-lift shortcut.
- False finish gate.
- Sheet tunnel.
- Box-stack switchback.

## Hazards And Beats

- False gates: visual misdirection, not unfair collision.
- Marble traps: rolling or timed obstacles.
- Sheet tunnel: visibility compression.
- Box switchbacks: tight technical turns.
- Shortcut: string-lift or hidden ramp path.

## Materials And Mood

- Materials: clean cardboard, polished trunk wood, cloth sheet, string, toy plastic prank pieces.
- Lighting: clean attic shafts, warm but uncanny, crisp shadows.
- Palette: cardboard tan, cloth white, trunk brown, Popper purple accents.
- Audio mood: creaks, box falls, prank squeaks, toy piano stabs, quick whooshes.

## Meshy Prompts

- "Stylized toy-scale attic prank trunk maze for an arcade kart track, pristine clean attic storage, false gates and hidden ramps, playful uncanny mood, no text, no humans, modular GLB-friendly."
- "Toy racing sheet tunnel and box-stack switchback set, clean attic, prank faction modifications, readable driving path, no logos, no text."
- "String-lift shortcut landmark for toy kart racing, clean attic beams, dangling string mechanism, hidden ramp route, low-poly game asset."

## Blender Tasks

- Split trunk maze, sheet tunnel, box stacks, false gates, and string-lift pieces.
- Keep sheet geometry simple and collision separate.
- Create alternate material variants for real vs fake gates without text.
- Make box stacks modular for route iteration.
- Keep string mechanisms low-poly and pivot-ready if animated.

## Godot Build Notes

- Keep misdirection readable and fair; false gates should teach, not randomly punish.
- Author hidden shortcuts with clear discovery cues after first lap.
- Use simple collision boxes for trunks and stacks.
- Avoid camera snags in sheet tunnel and tight switchbacks.
- If string-lift is interactive, keep animation deterministic and easy to test.
