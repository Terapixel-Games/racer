# Story Bible Concept Canon

`docs/concept_package.md` is the story bible for Toybox Dominion. This folder holds supporting concept canon that should stay beside that bible instead of living inside runtime track asset folders.

Use `concepts/stages/legacy_home_yard_courses/<course>/` for old `home_yard` course concepts and key art. Use `concepts/characters/` for roster concept art. Use `concepts/floor_plans/` for house and yard plan references.

Runtime stage generation should reference these paths as source canon. Do not move new concept-only art back under `assets/gameplay/tracks/` unless it becomes a runtime asset.
