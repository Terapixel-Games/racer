# Course Identity Reference

## Design Intent

Every public course keeps its original faction fantasy while being integrated into one coherent home stage.

## Shape Language

- Attic: pristine prank storage, trunks, sheets, strings, false gates.
- Playroom: champion arena, toy ring, ramps, block grandstands, trophy finish.
- Bedroom: warm protected refuge, bed ramp, blanket tunnel, lamp beacon, toy triage.
- Glam closet: mirror court, vanity runway, perfume mist, jewelry-box ramp, display pedestals.
- Outdoor playground: kinetic open run, slide drop, swing gate, rail shortcuts, chalk arrows.
- Garden: survival terrain, root gate, stone bridge, hose crossing, flower canopy.
- Sandbox: sand throne, bucket tunnel, fossil arch, shovel ramp, tribute pile.
- Kitchen: lawful court, sink gauntlet, cabinet court, utensil rails, tournament arch.

## Material Palette

Use each course palette from the original stage concepts while keeping house-wide lighting and scale consistent.

## Gameplay Readability Notes

Each course needs a clear start, first-turn read, route edge language, major landmark, stage interaction, and off-route dressing policy. Shortcuts must be discoverable and fair after first exposure.

## Asset Prompt Notes

Prompts should include the faction/course role and the exact route function: landmark, boundary, hazard, shortcut frame, start/finish language, or off-route decor.
