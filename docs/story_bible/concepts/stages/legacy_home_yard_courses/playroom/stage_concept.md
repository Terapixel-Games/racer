# Playroom Stage Concept

Stage: Playroom / Slammo  
Faction: The Arena  
Reference image: `stage_key_art.png`

## Identity

The Playroom is a loud champion arena. Slammo's faction turns silence into spectacle: title matches, toy crowd moments, banners, trophy lines, block grandstands, and high-energy race beats.

The room should feel pristine but overproduced, as if every toy conflict has been staged for a crowd.

## Lore Purpose

The Arena believes the house needs champions loud enough to make everyone feel alive again. Slammo gives toys noise, ritual, and shared events, but he also turns worth into performance.

The stage teaches that spectacle can replace despair, while leaving quiet toys behind.

## Visual Read

- Clean playroom with toy blocks, mats, shelves, banners, and arena dressing.
- Toy-scale route language: loops, ramps, ring platform, grandstands, trophy stretch.
- Faction modifications: champion signage without readable text, title belts, crowd toys, staged camera-like sightlines.
- Mood: loud, triumphant, competitive, toy-sports theatrical.

## Route Fantasy

The race should feel like a title match. Drivers circle an arena platform, hit champion ramps, dodge marble-machine hazards, smash through block-impact zones, and charge down a trophy finish stretch.

Shortcuts should feel like showboating: faster if executed cleanly, embarrassing if missed.

## Hero Landmarks

- Toy ring platform.
- Champion ramp.
- Block grandstands.
- Marble machine hazard.
- Trophy finish stretch.

## Hazards And Beats

- Marble hazards: rolling or timed obstacles.
- Block stacks: bumpable impact props.
- Arena loop: broad spectator-facing turn.
- Champion jump: high-visibility stunt beat.
- Shortcut: risky ramp line through grandstand edge.

## Materials And Mood

- Materials: foam mat, plastic blocks, toy vinyl, glossy trophy plastic, marble glass/plastic.
- Lighting: bright playroom overhead, arena-like accent lights.
- Palette: red, yellow, blue, trophy gold, Slammo red accents.
- Audio mood: crowd-like toy chants without vocals, spring ramps, plastic block impacts, victory stingers.

## Meshy Prompts

- "Stylized toy-scale playroom arena ring platform for an arcade kart track, clean toy sports spectacle, block grandstands, trophy finish lane, no text, no humans, modular GLB-friendly."
- "Toy racing marble machine hazard, clean playroom, rolling marble ramps and funnels, readable from race camera, no logos, no text."
- "Champion ramp and block grandstand set for toy kart racing, bright plastic, arena pageantry, no readable lettering."

## Blender Tasks

- Split ring platform, ramps, grandstands, marble machine, trophy props.
- Prepare marble hazard paths as separate Godot-controllable pieces.
- Keep crowd toys as simple silhouettes or low-cost dressing.
- Create collision helpers for ramps and arena edges.
- Avoid readable text on banners; use shapes and color.

## Godot Build Notes

- Author marble hazards as separate moving nodes if interactive.
- Keep arena loop wide and readable for high-speed pack racing.
- Use block stacks as optional bumpable hazards if physics budget allows.
- Ensure trophy finish stretch does not create visual clutter near finish line.
- Keep spectacle dressing outside primary driving collision.
