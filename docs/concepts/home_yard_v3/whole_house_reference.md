# Home Yard V3 Whole-House Reference

## Design Intent

Create a production-ready toy-scale home stage that combines the older public room and yard course concepts into one believable house, yard, street, and racing experience. The house should feel clean, inhabited, and intentionally arranged. Each course expresses a faction fantasy while still reading as part of the same home.

## Shape Language

Use a readable suburban family home: rectangular main floor, clear upper floor, believable gabled roof and attic volume, porch/front approach, service side, backyard outdoor rooms, and route-scale furniture. Course landmarks should be strong silhouettes, not generic blocks.

## Material Palette

Clean painted siding, warm interior wall paint, transparent glass, tile, wood trim, carpet, fabric, molded toy plastic, garden greens, clean sand, bright playground plastic, and Kenney/Meshy low-poly materials.

## Gameplay Readability Notes

The player must understand the house layout from cameras and route entrances: street/front yard arrival, main floor courses, upper floor courses, attic route, and backyard courses. Route edges need contrast, starts need immediate first-turn readability, and decorative assets must stay outside route corridors unless named hazards.

## Asset Prompt Notes

Meshy prompts should describe assets as low-poly, toy-scale, clean, readable from a kart racing camera, no text, no logos, no humans, GLB-friendly, and fitted to the named course fantasy.
