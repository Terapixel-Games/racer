# Backyard Reference

## Design Intent

The backyard should read as one coherent outdoor play and garden space. Outdoor playground, garden, and sandbox each need course identity, but they must share one yard system with fence/boundary logic, lawn, patio/deck, planting, grade, and movement.

## Shape Language

Outdoor rooms: kinetic playground, adaptive garden, dominant sandbox. Use lawn as connective tissue, fences/screens as containment, deck/patio as house transition, planting masses as boundaries, and large course landmarks as readable silhouettes.

## Material Palette

Grass, mulch, clean sand, concrete or deck wood, bright molded playground plastic, metal chains/rails, garden stones/roots/foliage, hose/water accents, bucket/shovel/fossil plastic.

## Gameplay Readability Notes

Outdoor courses need route starts and first turns that read against yard materials. Foliage must not block camera. Sandbox and playground structures need generous collision. Yard boundaries should prevent visual confusion about playable and forbidden space.

## Asset Prompt Notes

Meshy batches should cover one outdoor domain at a time: playground structure/swing/slide, garden root/stone/hose/flower landmarks, sandbox bucket/fossil/shovel/throne assets. Keep output low-poly and mobile-safe.
